# Flipkart_Scrapper
# Flipkart scrapping using BeautifulSoup 4 with flask 

## Features

There are so many features of this project are:

- Search term
- Product Name
- Rating
- Short Comment
- Main Comment

All the information are stored in mongodb as well as CSV file in the local system.
